# Gaussian-Delta-Visualizer
This project demonstrates the approximation of the Dirac Delta function using Gaussian functions. It provides a visual and intuitive understanding of how a narrow Gaussian distribution approaches an impulse function as the standard deviation decreases.

## 📌 Overview
This project demonstrates the concept of Dirac Delta function using Gaussian curves. As the standard deviation (σ) decreases, the Gaussian function becomes narrower and approaches an impulse-like behavior.

## 🧠 Concept
The Gaussian function:
\[
f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{x^2}{2\sigma^2}}
\]

As σ → 0, the curve approaches the Dirac Delta function.

## 🚀 Features
- Visualization of Gaussian curves
- Adjustable sigma (σ) values
- Comparison of multiple curves
- Educational tool for physics and signal processing

## 📊 Output
Plots show how narrowing Gaussian curves approximate an impulse function.
